protected

Builds a 2D Cartesian grid that encloses a given mesh.

To avoid numerical rounding the bounding box is enlarged by a small factor

Parameters
meshThe input mesh used to determine the boundaries.
n_cells_xThe number of grid cells along the x-axis.
stretchA factor to stretch the grid cells in the y-direction relative to the x-direction.
Returns
An array of two vectors, representing the grid line coordinates for the x and y axes.
