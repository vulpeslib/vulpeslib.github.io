Reads a 3D mesh from a VTU file.

This function reads an unstructured grid from a VTU file (a format from the VTK library) and creates a Mesh3D object from it. It handles different cell types like polyhedrons and tetrahedrons and can also read associated cell tags.

Template Parameters
FloatTypeThe floating-point type.
IndexTypeThe integer type.
Parameters
filenameThe path to the VTU file.
check_vtxs_uniqueA boolean flag (not used in the current implementation).
add_tagsA boolean flag to indicate if cell tags should be read.
Returns
A new Mesh3D object.

Creates a rectangular cuboid mesh.

This function generates a structured mesh of a rectangular cuboid (a box) defined by two opposite corners. The mesh can be composed of either hexahedral (cube) elements or tetrahedral elements.

Template Parameters
FloatTypeThe floating-point type.
IndexTypeThe integer type.
Parameters
corner_aThe coordinates of the first corner of the cuboid.
corner_bThe coordinates of the opposite corner of the cuboid.
n_divisionsThe number of divisions along each axis (x, y, z).
tetra_elementsA boolean flag to determine if the elements should be hexahedrons (false) or if each hexaedron should be cut in six tetrahedrons (true).
Returns
The generated Mesh3D object.
