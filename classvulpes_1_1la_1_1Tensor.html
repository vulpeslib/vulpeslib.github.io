Detailed Description

Ultra-lightweight wrapper for PETSc Vec with Eigen mapping. Loads only the data and maps it using externally provided cell IDs. For now it considers only tensors which are stored in a flatten array.

Construct a Tensor by loading data from HDF5. The class can manage only cell constant tensors.

Parameters
filenamePath to the .h5 file.
dataset_nameInternal HDF5 path (e.g., "/DTI/vector_0").
Note
The tensor must be stored in a flatten way and the values must be stored using the cell id order. The starting location in the flatten version of the tensor inside the file for cell id \( i \) is given by \( i * Dim * Dim \).
