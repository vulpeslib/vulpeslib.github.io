Detailed Description

template<std::size_t Dim>
class vulpes::fe::FeSpaceVEM< Dim >

Virtual Element Method FE Space on polytopal elements.

Warning
This FE Space is still experimental and not all features are implemented.

The idea of VEM finite element space is that it is duty of this class to compute the extra matrices needed during the assembly of the global system (like the matrices D, B and the moment projection). To fullfill this task the class can inject in an integrator extra weak forms (FeExpressions) that are used to compute these matrices.
