Detailed Description

template<typename T, size_t Dim>
class vulpes::geometry::KMeansNaive< T, Dim >

Implementation of the Lloyd algorithm (also known as the naive KMeans) for clustering.

This class provides a straightforward implementation of the classic Lloyd's algorithm to solve the KMeans clustering problem. It iteratively assigns each data point to the nearest centroid and updates centroids as the mean of their assigned points, converging to a local minimum of the KMeans objective. The class inherits from KMeansBase and uses its interface and types.
