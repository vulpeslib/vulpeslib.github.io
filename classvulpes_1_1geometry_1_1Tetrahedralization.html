Detailed Description

Performs tetrahedralization of a 3D cell using the TetGen library.

This class generates a tetrahedral mesh for a given 3D cell and provides access to the resulting points and tetrahedra. It leverages the TetGen library [http://wias-berlin.de/software/tetgen/] under the hood for 3D computational geometry operations.
