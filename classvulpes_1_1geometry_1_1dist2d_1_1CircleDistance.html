Detailed Description

template<typename FloatType>
class vulpes::geometry::dist2d::CircleDistance< FloatType >

Signed distance function for a circle in 2D space.

This class computes the signed distance between a point and a circle defined by its center and radius.

Template Parameters
FloatTypeThe type of floating-point value.
