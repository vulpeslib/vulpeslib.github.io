Detailed Description

template<typename MeshDerived>
class vulpes::geometry::MeshExporterSubmesh< MeshDerived >

Exports a mesh as multiple VTK grids, one for each cell.

This class extends MeshExporterBase to create a separate vtkUnstructuredGrid for each cell (or element) of the original mesh. This is useful for visualizing results that are defined on a per-cell basis, especially when solving problems with discontinous methods.

Template Parameters
MeshDerivedThe derived mesh class.
