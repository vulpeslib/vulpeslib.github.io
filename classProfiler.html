Detailed Description

Collects high-resolution timing data for labeled code sections.

When NPROFILE is not defined, start/stop calls record durations in nanoseconds under a given label. Intervals support multiple start/stop pairs to measure repeated events. Accumulated times can be printed or cleared. A JSON dump method provides machine-readable output.
