template<std::size_t Dim>
auto vulpes::assemble::dg::createPenaltyExpressionHarmonic ( std::size_t order,
double gamma )

Create a penalty expression for DG methods using the harmonic mean between the two cells.

Template Parameters
DimDimension of the problem
Parameters
orderOrder of the polynomial basis
gammaPenalty parameter
Returns
Finite element expression containing the penalty value
Note
The penalty is computed as gamma * order^2 * Cinv * (h_f / |K|), where Cinv = |K| / Kb, h_f is the face diameter, |K| is the cell measure and Kb is the K flat introduced in Cangiani, Dong, H. Georgoulis, Houston (2017). The final penalty is the harmonic mean between the two cells sharing the face.

template<std::size_t Dim>
auto vulpes::assemble::dg::createUpwindFluxConstant ( double vx,
double vy = 0.0,
double vz = 0.0 )

Legacy constant-velocity upwind helper.

Template Parameters
DimDimension of the problem
Parameters
vxX-component of velocity
vyY-component of velocity (only for 2D)
vzZ-component of velocity (only for 3D)
Returns
Finite element expression containing the upwind flux coefficient
Note
The upwind flux is computed as 0.5 * (|velocity.n| + velocity.n), which evaluates to velocity . n when velocity . n > 0 (outflow) and 0 when velocity . n < 0 (inflow). This implements the standard upwind flux for first-order hyperbolic problems following Brezzi, Marini, Suli (2004).
Prefer createUpwindFlux<Dim>(velocity_func) for the unified q-point based path used across advection assembly and tests.
