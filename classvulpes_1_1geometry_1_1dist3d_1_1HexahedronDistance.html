Detailed Description

template<typename FloatType>
class vulpes::geometry::dist3d::HexahedronDistance< FloatType >

Signed distance function for a hexahedron (cuboid) in 3D space.

This class computes the signed distance between a point and a hexahedron defined by two corner points.

FloatTypeThe type of floating-point value.
