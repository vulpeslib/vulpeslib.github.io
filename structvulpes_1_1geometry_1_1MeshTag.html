Detailed Description

template<std::size_t Dim, typename IndexType>
struct vulpes::geometry::MeshTag< Dim, IndexType >

A structure to store tags for different parts of a mesh.

Tags are like labels that you can attach to mesh entities (like vertices, facets, or cells). This is useful for identifying specific regions, like boundaries or different materials.

Template Parameters
DimThe dimension of the mesh entities being tagged.
IndexTypeThe type used for tagging.
