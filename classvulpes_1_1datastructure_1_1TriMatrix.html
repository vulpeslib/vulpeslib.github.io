Detailed Description

template<bool Triangular, bool Tensor, typename DataType, typename IndexType, typename StorageType = std::vector<DataType>>
class vulpes::datastructure::TriMatrix< Triangular, Tensor, DataType, IndexType, StorageType >

A class that represents a triangular or square matrix, with optional support for tensor-like multi-dimensional data.

The TriMatrix class is a templated matrix data structure that can store elements in either triangular or full (square) format. Namely it offers a coalescent representation for data when you know that the indexes of the tensor i, j satisfy i + j < tri_dim. It also supports third-order tensor data if enabled through template parameters.

Template Parameters
TriangularSpecifies whether the matrix is triangular. If true, only the upper triangular part is stored.
TensorSpecifies whether the matrix operates as a tensor with additional dimensions.
DataTypeThe data type of elements stored in the matrix (e.g., double, float).
IndexTypeThe data type for indexing the matrix (typically int or size_t).
StorageTypeThe container where to save the data. By default an std::vector, other used options are a const pointer, indicating that this is an not-owning view of some data
