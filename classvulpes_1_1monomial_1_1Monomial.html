Detailed Description

template<std::size_t Dim, typename IndexType, typename CoefType, bool IsScaled>
class vulpes::monomial::Monomial< Dim, IndexType, CoefType, IsScaled >

Class representing a (possibly scaled) monomial.

This class offers the possibility to work with monomials and scaled monomials, that are monomials of the form \((\frac{x - x_0)}{h})^\alpha\).
