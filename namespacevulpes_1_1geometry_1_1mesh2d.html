Creates a new mesh by intersecting an existing mesh with a Cartesian grid.

This function "cuts" the boundary of the input mesh with the grid lines, resulting in a new, less complex mesh. Small fragments created during intersection with the boundary can be merged into larger cells based on a threshold.

Parameters
meshThe original mesh to intersect.
gridsThe Cartesian grid lines (x and y coordinates).
agglomeration_thresholdA threshold (relative to cell area) below which small cells are merged.
Returns
A new Mesh2D object representing the intersected mesh.

Merges two separate meshes into a single mesh.

This function combines the vertices and cells from two meshes. It can identify and merge vertices that are very close to each other, based on the specified tolerances.

Warning
If the boundaries are not completely matching the merging will fails. This does not mean that the tolerances must be zero but the discretization of the matching boundary must be the same (e.g., you can merge two edges (0, 0)-(1, 0) and (eps, eps)-(1-eps, -eps) but you cannot merge (0,0)-(0.5,0)-(1,0) with (0,0)-(1,0))
Parameters
mesh_aThe first mesh.
mesh_bThe second mesh.
atolAbsolute tolerance for merging vertices.
rtolRelative tolerance for merging vertices.
Returns
A new Mesh2D object containing both input meshes.
