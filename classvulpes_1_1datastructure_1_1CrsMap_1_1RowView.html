Detailed Description

template<typename DataType, typename IndexType, IndexType RowSize = IndexType(-1)>
template<bool Mutable>
class vulpes::datastructure::CrsMap< DataType, IndexType, RowSize >::RowView< Mutable >
Template Parameters
MutableBoolean flag to indicate whether the row data is mutable.

This class provides a view over a single row in the CRS map. The row's elements can be accessed via iterators or array-style indexing.
