Factory function for FeExpressionGradContr with gradient on TRIAL (adjoint form, GradOnTest=false).

Builds: sum_I int_K v_I * F . grad(u_I) dK

Assembles the adjoint stiffness T_{ij} = int_K phi_i * F . grad(phi_j) dK directly – no integration-by-parts, no face terms.

Typical use: auto wf = fe::grad_contr(c_expr, v, fe::grad(u));

Factory function for FeExpressionGradContrBlock with gradient on TRIAL (adjoint form, GradOnTest=false).

Builds the bilinear expression v_k * F . grad(u_l) for the single (TestComp, TrialComp) component block. TestComp and TrialComp must be specified explicitly; all other template parameters are deduced from v and grad_u.

Assembles: T_{ij} = int_K phi_i * F . grad(phi_j) dK This is the transpose of the test-gradient stiffness – no IBP, no face terms.

Typical use (off-diagonal block k=0, l=1): auto wf = fe::grad_contr_block<0, 1>(A_expr, v, fe::grad(u));

Factory function for FeExpressionInnerBlock.

Builds the bilinear expression \( c(x) \, u_l \, v_k \) for the single (TestComp, TrialComp) block. TestComp and TrialComp must be specified explicitly; the remaining template parameters are deduced from u and v.

The test argument v may be either a regular vector test function ( FeExpressionFunc<...,true> ) or its neighbour-side wrapper ( FeExpressionNeighborFunc<...,true> obtained via fe::neigh ).

Typical use (off-diagonal coupling between two field components, e.g. one entry of a flux Jacobian split):

auto wf_diag = fe::inner_block<0, 1>(coeff_expr, u, v);
auto wf_neigh = fe::inner_block<0, 1>(coeff_expr, u, fe::neigh(v));

2D perpendicular rotation of a 2-vector coefficient expression.

Returns the 90-degree counterclockwise rotation: perp([u, v]) = [-v, u]

Implemented as make_vec(-spatial_component<1>(e), spatial_component<0>(e)). No new class is needed; this is a free factory that composes existing expressions. Only defined for Dim=2 and expressions with OutputType = Eigen::Vector2d.

Typical use (scalar weighting of a perpendicular 2D vector field):

auto w_perp = fe::perp(fe::make_vec(p_proj, q_proj));
auto wf = fe::inner(c * w_perp, V);
