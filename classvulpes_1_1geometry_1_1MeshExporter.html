Detailed Description

template<typename MeshDerived>
class vulpes::geometry::MeshExporter< MeshDerived >

Exports a single mesh as a single VTK grid.

This class extends MeshExporterBase to handle a single mesh and convert it into a single VTK unstructured grid. It provides methods to attach both vertex and cell data to this grid. The constructor automatically builds the grid based on the mesh's dimensionality.

Attaches a list of vertex data to the VTK grid.

The data is attached to the points (vertices) of the grid, with each row of the matrix being a separate data array (in column major format). That is, data is contiguous across the same vertex list.

Parameters
labelThe base name for the data arrays. An index will be appended for each row.
