Detailed Description

template<std::size_t Dim, typename FloatType>
class vulpes::qrule::QuadratureLibrary< Dim, FloatType >

A library for managing and generating quadrature rules for numerical integration.

The QuadratureLibrary class provides a mechanism to lazily generate and cache quadrature rules for various element types and integration orders. The rules are generated for a specific spatial dimension and can be accessed using a key consisting of the element type and order.

The coefficient of the polynomials are computed exactly by using mpq_class (a class that represent a fraction in arbitrary precision implemented in GNU GMP) using the recursive formula of Legendre polynomials (Bonnet's recursion formula). Evaluation of the polynomials is done using the mpf_class to handle floating point in arbitrary precision. Nodes and points are computed using Newton's iterations

For the n-dimensional simplex we employ the following strategy:

  • Tabulated literature quadrature rules for low order. We choose rules that have positive weights and internal points. Different sources consider different reference triangles, thus a tranformation may be needed. Original points are in the repo as raw files. At the following links you may find a review of the quadrature rules http://web.archive.org/web/20240415021756/https://www.math.unipd.it/~alvise/SETS_CUBATURE_TRIANGLE/rules_triangle.html, https://github.com/MatteoCaldana/legacy-quadpy
  • General rules for high-order. The tradeoff is that literature quadrature have less quadrature points, thus are cheaper, but (by nature) less general.
Template Parameters
DimThe spatial dimension of the quadrature rules (1, 2, or 3).
