Detailed Description

template<std::size_t Dim, typename = std::enable_if_t<(Dim == 2) || (Dim == 3)>>
class vulpes::geometry::Simplexify< Dim, typename >

A templated class that simplifies a geometric cell into simplices (triangles or tetrahedra).

This class provides a unified interface for performing either 2D triangulation or 3D tetrahedralization, depending on the dimension (Dim) of the input cell. It leverages the Triangulation class for 2D cases and the Tetrahedralization class for 3D cases.

Template Parameters
DimThe dimension of the input cell. Must be 2 (for 2D) or 3 (for 3D).
EnableSFINAE helper to restrict the template parameter to valid dimensions.
