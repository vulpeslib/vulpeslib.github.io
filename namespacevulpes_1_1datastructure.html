template<typename DataType >
CrsMapUncompressedVariant< DataType > vulpes::datastructure::getUncompressedCrsMapWithStaticStorageGreaterThan ( std::size_t  static_size)

Function that builds an UncompressedCrsMap with at least a certain static storage, up to 8192 elements.

It may happen that you do not know at compile time a good static size to save the uncompressed data of your map. This function builds a variant with at least a certain amount of static storage up to 8192 elements.

Warning
the overhead of dispatching an std::visit to the std::variant might be significant
Returns
std::variant containing an UncompressedCrsMap

The function writes the CrsMap row by row, displaying each row's index and contents. The format is:
