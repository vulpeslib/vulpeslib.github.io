Detailed Description

template<typename T, size_t Dim>
class vulpes::geometry::KMeansBase< T, Dim >

Base class template for KMeans clustering algorithms.

This class defines the shared interface and common attributes for all KMeans-based clustering algorithms, such as KMeansNaive. It provides type definitions, constructors, and accessors for cluster labels, centroids, and algorithm parameters. Derived classes should implement the clustering logic by overriding the pure virtual fit() method.
