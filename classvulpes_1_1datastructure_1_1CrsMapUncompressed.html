Detailed Description

template<typename DataType, std::size_t N = 16>
class vulpes::datastructure::CrsMapUncompressed< DataType, N >

A class that behaves like a vector of vectors, providing automatic row range checking.

This class is designed to store (staggered) 2D data efficiently using a vector of SmallVector objects. It provides automatic range checking of the rows, ensuring valid access to the elements. A key performance improvement over a standard vector of vectors is the usage of coalescent memory for rows, up to a threshold specified by template parameter N. Each row is implemented as a SmallVector, which can dynamically switch between static and dynamic storage based on the size of the row.

Template Parameters
DataTypeThe type of data to be stored in the 2D structure.
NThe size threshold after which rows are stored dynamically. Defaults to 16.
Note
: if the number of columns is fixed, it is better to use another datastucture.
