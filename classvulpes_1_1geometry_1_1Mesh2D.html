Detailed Description

template<typename FloatType, typename IndexType>
class vulpes::geometry::Mesh2D< FloatType, IndexType >

A class for a 2D mesh, inheriting from MeshBase.

This class provides a specific implementation for a two-dimensional mesh composed of polygonal cells. It defines strict ordering rules to ensure consistency, which is important for many numerical algorithms.

Mesh Properties and Ordering Rules:

  • Cell to Vertex: Vertices defining a cell are always ordered counter-clockwise (CCW).
  • Cell to Facet: Facets (edges) of a cell are ordered to follow the CCW sequence of vertices.
  • Facet to Vertex: The two vertices of a facet are always ordered by their index, from smallest to largest.
  • Facet to Cell: For an interior facet separating two cells, the first cell in the map is the one whose CCW orientation is consistent with the facet's direction. The second cell is the neighbor. For a boundary facet, one of the cell indices will be UNDEF.
Template Parameters
FloatTypeThe data type for floating-point numbers (e.g., float or double).
IndexTypeThe data type for indices (e.g., int or std::size_t).
