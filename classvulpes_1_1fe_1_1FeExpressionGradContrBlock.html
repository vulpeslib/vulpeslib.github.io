Detailed Description

template<std::size_t TestComp, std::size_t TrialComp, std::size_t IndexBegin, std::size_t IndexEnd, std::size_t Dim, typename FeSpace, typename FuncExpr, bool GradOnTest = true>
class vulpes::fe::FeExpressionGradContrBlock< TestComp, TrialComp, IndexBegin, IndexEnd, Dim, FeSpace, FuncExpr, GradOnTest >

Bilinear expression for a single (test, trial) component block. Two forms selected by GradOnTest:

GradOnTest=true (default): int_K F(x) . grad(v_k) * u_l dK GradOnTest=false (adjoint): int_K v_k * F(x) . grad(u_l) dK

Activates exactly one block of the local matrix identified by (TestComp, TrialComp). All other (J, I) pairs return void and are skipped by the integrator.

When TestComp == TrialComp this is a diagonal block (same as one component of FeExpressionGradContr). When TestComp != TrialComp this is an off-diagonal block, expressing coupling between two distinct components.

A common use case is a linearized hyperbolic system where the flux Jacobian has off-diagonal entries A_{kl}(x) coupling test component k with trial component l: wf_kl = fe::grad_contr_block<k, l>(A_kl_expr, fe::grad(v), u);

The trial-gradient form (GradOnTest=false) assembles the adjoint stiffness: T_{ij} = int_K phi_i * F . grad(phi_j) dK which equals the transpose of the test-gradient form – no IBP, no face terms. Needed when the adjoint stiffness T_{ij} = int_K phi_i * F . grad(phi_j) dK is required directly: wf_kl = fe::grad_contr_block<k, l>(F_expr, v, fe::grad(u));

Note: evaluatePoly / evaluatePolyCoef are not implemented. This expression supports the simplexify (quadrature-based) integrator only.

Evaluate the (J, I) block contribution at quadrature point q.

J = D % nblocks is the trial component (col block index). I = D / nblocks is the test component (row block index). The integrator accumulates: row = j_basis * nblocks + I col = i_basis * nblocks + J This expression fires only when J == TrialComp and I == TestComp. All other (J, I) pairs return void.

GradOnTest=true: F(x_q) . grad(phi_i) * phi_j (gradient on test i) GradOnTest=false: phi_i * F(x_q) . grad(phi_j) (gradient on trial j)
