Detailed Description

A fully compressed representation of a gathered monomial GatheredMonomial for face integrals.

This struct uses integer IDs to reference all parts of a gathered monomial, enabling efficient storage and lookup in precomputed tables. For details on the meaning of the terms, see GatheredMonomial
