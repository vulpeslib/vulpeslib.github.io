Computes the 2D monomial exponents for a given linear index.

Template Parameters
TThe integral type used for the return array elements.
DummyA dummy template parameter used for SFINAE (Substitution Failure Is Not An Error).
Parameters
nThe linear index for which the 2D monomial exponents are to be computed.
Returns
A std::array<T, 2> containing the monomial exponents \((i, j)\) corresponding to the input index \(n\).

This function calculates the exponents \(i\) and \(j\) for the monomial \(x^i y^j\) that corresponds to the linear index \(n\) in a 2D monomial ordering scheme. The function assumes the index \(n\) is non-negative and within the range where the floating-point precision of float is sufficient to handle the calculations. It uses a quadratic formula to determine the level of the monomial and then computes the specific exponents within that level. The triangle up to the \(s\)-th level contains \(s(s+1)/2\) elements

Example

  • For \(n = 0\), the result will be \((0, 0)\).
  • For \(n = 1\), the result will be \((1, 0)\).
  • For \(n = 2\), the result will be \((0, 1)\).

Parameters
nThe linear index for which the 3D monomial exponents are to be computed.
Returns
A std::array<T, 3> containing the monomial exponents \((i, j, k)\) corresponding to the input index \(n\).

This function calculates the exponents \(i\), \(j\), and \(k\) for the monomial \(x^i y^j z^k\) that corresponds to the linear index \(n\) in a 3D monomial ordering scheme. The function handles the case when \(n = 0\) by returning \((0, 0, 0)\), and for \(n > 0\), it uses a cubic root approximation and Newton's method to determine the pyramid level. Namely, the pyramid up to the \(s\)-th level contains \(s(s+1)(s+2)/6\) elements The specific exponents within that level are then computed using the getMonomial2D function for the residual index.

Example

  • For \(n = 0\), the result will be \((0, 0, 0)\).
  • For \(n = 1\), the result will be \((1, 0, 0)\).
  • For \(n = 2\), the result will be \((0, 1, 0)\).
  • For \(n = 3\), the result will be \((0, 0, 1)\).
