Detailed Description

template<typename T, std::size_t Dim>
struct vulpes::qrule::QuadraturePoint< T, Dim >

Represents a single quadrature point in a specified dimension.

A quadrature point consists of its coordinates and an associated weight used for numerical integration.

Template Parameters
TThe data type for the coordinates and weight (e.g., float, double).
DimThe spatial dimension of the quadrature point.
