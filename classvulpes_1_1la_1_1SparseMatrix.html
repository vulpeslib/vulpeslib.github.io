◆ pinDof()

void vulpes::la::SparseMatrix::pinDof ( PetscInt dof_idx,
PetscScalar value = 1.0 )

Pin a degree of freedom to enforce a constraint This performs symmetric elimination: sets row and column of the specified DOF to zero, except the diagonal which is set to 1.0. This removes a DOF from the system and is useful for enforcing Dirichlet BCs or removing null space (e.g., constant modes in periodic problems). The corresponding RHS entry should be set to the desired value.

Note
Before calling this method, the matrix object must be in compressed state, e.g. by means of SparseMatrix::compress().
Parameters
dof_idxThe global index of the DOF to pin
valueThe diagonal value to set (default 1.0)
