inline

Accumulates the unnormalized normal of a polygon using Newell's method.

This function calculates the geometric normal of an ordered, nearly-planar polygon and adds it to the provided output buffer out_normal. It is robust for non-convex polygons and operates on raw pointer data for high performance.

Note
This is an accumulator function. It does not initialize out_normal to zero before computation. The caller is responsible for zeroing the output buffer if they are not accumulating results.
Template Parameters
FloatTypeThe floating-point type for coordinates (e.g., float, double).
IndexTypeThe integer type for vertex indices (e.g., int, uint32_t, size_t).
Parameters
[in]vertices_dataPointer to a contiguous array of vertex coordinates, laid out as [x0, y0, z0, x1, y1, z1, ...].
[in]face_indices_dataPointer to an array of vertex indices that define the polygon in counter-clockwise order.
[in]num_face_verticesThe number of vertices in the polygon face.
[in,out]out_normalPointer to a buffer of 3 FloatTypes. The calculated normal components are added to the existing values in this buffer.
