Detailed Description

Parallel-aware output stream.

pcout behaves like std::cout, but:

  • prints only on the MPI root rank (if MPI is initialized)
  • supports log levels (ERROR, INFO, DEBUG)
  • enables DEBUG output automatically in Debug builds
