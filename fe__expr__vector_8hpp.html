Detailed Description

Bilinear/linear expressions for vector-valued FE spaces.

Provides composable weak-form building blocks used to assemble systems whose trial and test functions are vector-valued (multi-component) FE functions:

  • FeExpressionGradContr / grad_contr: diagonal sum \( \sum_k \int_K F \cdot \nabla v_k \, u_k \, dK \).
  • FeExpressionGradContrBlock / grad_contr_block: single off-diagonal \( \int_K F \cdot \nabla v_k \, u_l \, dK \) for one (TestComp,TrialComp) pair (volume term).
  • FeExpressionInnerBlock / inner_block: single off-diagonal \( \int c \, v_k \, u_l \, dS \) for one (TestComp,TrialComp) pair. Used for matrix-valued face fluxes in linearised hyperbolic systems.

All block expressions integrate over the (TestComp,TrialComp) pair only; other (I,J) blocks return void() and are skipped by the integrator. Mirrors the existing iteration pattern in WeakFormIntegratorSimplexify so no integrator changes are required to support matrix-valued couplings between components.
