Detailed Description

template<typename T, std::size_t N>
class vulpes::datastructure::SmallVector< T, N >
Template Parameters
TThe type of the elements to be stored.
NThe maximum size before switching to dynamic storage.

This class behaves like a vector but uses coalesced memory for small sizes (up to N elements). Beyond that, it switches to dynamic memory.

Note
: iteration on the elements via iterators might be more performant than access with operator[] since the latter has a size check at each call
