Detailed Description

template<std::size_t Dim, typename FeSpace, typename DofHandler, std::size_t OutputDim = 1>
class vulpes::fe::FeExactExpressionFeProjection< Dim, FeSpace, DofHandler, OutputDim >

Class that wraps the projected of a given vector of coefficients on the FE space.

Note
This is used only for computing the projection for the exact solution when the L2 error is computed.
