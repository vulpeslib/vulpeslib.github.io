Detailed Description

template<std::size_t IndexBegin, std::size_t IndexEnd, std::size_t Dim, typename FeSpace, typename FuncExpr>
class vulpes::fe::FeExpressionGradContr< IndexBegin, IndexEnd, Dim, FeSpace, FuncExpr >

General bilinear expression: F . grad(v_I) * u_I, summed over diagonal component blocks (I == J only).

Computes the volume contraction: sum_I int_K F(x) . grad(v_I) * u_I dK

where F is a Dim-vector expression (e.g., an advection velocity, a flux direction, or any spatially-varying Dim-vector field), v_I is the I-th component of the vector test function, and u_I is the I-th component of the vector trial function.

Only diagonal component blocks (trial component J == test component I) are nonzero. Off-diagonal blocks return void and are skipped by the integrator, producing a block-diagonal structure in the local matrix.

Typical use (advection volume term, adjoint form, gradient on test): auto wf = fe::grad_contr(c_expr, fe::grad(v), u);

Note: evaluatePoly / evaluatePolyCoef are not implemented. This expression supports the simplexify (quadrature-based) integrator only, not the qfree (quadrature-free) integrator.

Evaluate the (J, I) block contribution at quadrature point q.

Returns F(x_q) . grad(phi_i) * phi_j when J == I and I is in the component range [IndexBegin, IndexEnd). Returns void for J != I (off-diagonal blocks, skipped by the integrator) and for components outside the declared range.

The result is a scalar (double), consistent with the integrator accumulation: m_integrals(j*nblocks+J, i*nblocks+I, id) += result*Jdx.

Template Parameters
JTrial component index (compile-time constant).
ITest component index (compile-time constant).
