Detailed Description

template<typename T, std::size_t Dim>
struct vulpes::qrule::QuadratureRule< T, Dim >

Represents a quadrature rule consisting of a set of points and associated weights.

A quadrature rule is used to approximate integrals over a domain. It contains a collection of quadrature points, each with a coordinate and a weight, and the polynomial degree of accuracy.

Template Parameters
TThe data type for the coordinates and weights (e.g., float, double or mpf_float).
DimThe spatial dimension of the quadrature rule.
