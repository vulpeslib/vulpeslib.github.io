Detailed Description

template<typename MeshType>
class vulpes::geometry::MeshMetrics< MeshType >

A class template that can be used to compute various mesh metrics (CircleRatio, AreaToPerimeterRatio, UniformityRatio, VolumeDifference)

The MeshMetrics class template provides methods to compute and access various geometric quality metrics for each cell in a mesh, such as circle ratio, area-to-perimeter ratio, uniformity factor, and volume difference. The metrics to be computed can be selected on costruction via the MetricsInitOptions enum, allowing flexible and efficient analysis of mesh quality. Metrics are computed using the cell class and are stored internally for fast access. The class supports both 2D and 3D meshes, and is compatible with different floating-point and index types.

Template Parameters
MeshTypeThe particular mesh type on which the metrics have to be computed on (e.g. Mesh2D<double, int32_t>, etc.)
