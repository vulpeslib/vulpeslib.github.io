Add a scalar vector into one stride of a blocked vector.

Deduces the block size from the size ratio blocked/scalar and calls VecSetBlockSize before scattering, so this works even when blocked_vec was constructed without an explicit block size (e.g. from a size-only constructor).

Extract one scalar component from a blocked vector.

Deduces the block size from the size ratio blocked/scalar and calls VecSetBlockSize before gathering, so this works even when blocked_vec was constructed without an explicit block size (e.g. from a size-only constructor such as an RK stage scratch vector). Symmetric counterpart to addComponent (which uses VecStrideScatter).

Template Parameters
DofHandlerTypeType of the DOF handler used to build output vector
Parameters
blocked_vecInput blocked vector
dof_handlerDOF handler for the extracted scalar vector
fComponent index to extract
Returns
Scalar vector containing component f of the blocked input
