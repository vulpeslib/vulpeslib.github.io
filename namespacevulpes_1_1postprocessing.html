const std::vector< double > &  h 
)

Estimate convergence orders from a sequence of errors and mesh sizes.

For consecutive refinement levels i and i+1, the order is computed as log(err[i] / err[i+1]) / log(h[i] / h[i+1]).

Parameters
errError values, one per refinement level (size >= 2).
hMesh diameters corresponding to each level (same size as err).
Returns
Vector of orders, size = err.size() - 1.
Exceptions
std::invalid_argumentif sizes are mismatched or smaller than 2.

inline

Print a unified convergence table for one or more named error series.

Each entry in series is a (column_name, error_vector) pair. Convergence orders are computed internally between consecutive refinement levels. Format: scientific notation (3 sig-figs) for errors, fixed (2 d.p.) for orders, "---" on the first row where no order is available.

Parameters
h_vecMesh diameters, one per refinement level.
seriesNamed error series to display. All vectors must match h_vec.
output_dirIf non-empty, the table is also saved as a CSV in this directory (created if it does not exist).
filename_stemBase name of the CSV file without extension.
