Detailed Description

template<typename FloatType, int32_t Dim, typename D1, typename D2>
class vulpes::geometry::UnionDistance< FloatType, Dim, D1, D2 >

Represents the union of two signed distance functions.

This class computes the signed distance between a point and the union of two shapes, where the distance to the union is defined as the minimum of the distances to the individual shapes.

Template Parameters
FloatTypeThe type of floating-point value.
DimThe dimensionality of the space (e.g., 2D or 3D).
D1The first distance function (e.g., RectangleDistance, CircleDistance).
D2The second distance function.
