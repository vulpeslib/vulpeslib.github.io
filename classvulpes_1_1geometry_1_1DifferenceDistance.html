Detailed Description

template<typename FloatType, int32_t Dim, typename D1, typename D2>
class vulpes::geometry::DifferenceDistance< FloatType, Dim, D1, D2 >

Represents the difference of two signed distance functions.

This class computes the signed distance between a point and the difference of two shapes, where the distance to the difference is defined as the maximum of the negative distance to the second shape and the distance to the first shape.

Template Parameters
FloatTypeThe type of floating-point value.
DimThe dimensionality of the space (e.g., 2D or 3D).
D1The first distance function.
D2The second distance function.
