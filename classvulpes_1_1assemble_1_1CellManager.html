Detailed Description

template<typename MeshType, typename FeSpaceType>
class vulpes::assemble::CellManager< MeshType, FeSpaceType >

Manages geometrical informations about the cell and its mapping to the reference element.

This class constructs lazily all the cells of a mesh (in a mesh all cells are represented implicitly). This requires a lot of extra memory, but the having contiguous information (e.g., for vertices) significantly improves the speed of the code. The cells are initialized builinding all the extra informations required by the finite element space. The cells are then mapped to the reference element and the mapping is stored. This class also offers the possibility to query the neighbour of the cell in local cell enumeration, which is more costly to do in global mesh enumeration
