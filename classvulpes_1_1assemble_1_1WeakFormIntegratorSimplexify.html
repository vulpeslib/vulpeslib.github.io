Detailed Description

template<typename MeshType, typename FeExpressionTuple, typename FeExpressionFaceTuple = std::tuple<>>
class vulpes::assemble::WeakFormIntegratorSimplexify< MeshType, FeExpressionTuple, FeExpressionFaceTuple >

Integrator that simplexifies cells and integrates weak forms on them.

The WeakFormIntegratorSimplexify class extends WeakFormIntegratorBase and provides an integration algorithm that:

  • subdivides a possibly non-simplex cell into simplexes (via geometry::Simplexify),
  • maps quadrature points from a reference simplex to each physical simplex,
  • evaluates finite-element basis and expressions at quadrature points,
  • accumulates volume and boundary integrals into local element matrices / vectors.

The strategy avoids requiring the mesh cells themselves to be simplexes by working on the simplex subdivision explicitly. Boundary integrals are handled similarly by subdividing faces into lower-dimensional simplexes (edges in 2D, triangles in 3D).

The integrator therefore:

  1. computes the affine map \(A_\ell,b_\ell\) for each simplex,
  2. rescales quadrature weights by \(|\det(A_\ell)|\),
  3. evaluates FE basis and expressions at mapped quadrature points,
  4. accumulates contributions into bilinear forms (matrices) and linear functionals (rhs) using provided block structure (s_n_blocks).
