Detailed Description

template<typename MeshType, typename FeSpaceType, typename ExprType>
class vulpes::assemble::ScalarExpressionIntegrator< MeshType, FeSpaceType, ExprType >

Specialized integrator for scalar expressions using simplification.

This class integrates scalar FE expressions (e.g., f(x), (u_h - u_exact)^2) directly at quadrature points. It reuses the simplification and quadrature infrastructure from WeakFormIntegratorSimplexify but evaluates expressions as pure scalars instead of assembling weak forms.

Structure mirrors WeakFormIntegratorSimplexify:

  • Simplifies cells into simplexes
  • Maps quadrature points with proper Jacobian scaling
  • Evaluates expressions at quadrature points
  • Accumulates integral = sum_q expr(q) * weight(q)
