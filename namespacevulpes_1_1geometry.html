template<int32_t Dim, typename = std::enable_if_t<(Dim == 2) || (Dim == 3)>>
using vulpes::geometry::VoronoiCell = typedef std::conditional_t< Dim == 2, voro::voronoicell_neighbor_2d, voro::voronoicell_neighbor>

Alias for the Voronoi cell type based on the dimensionality (2D or 3D).

This template alias defines the appropriate type of Voronoi cell (either 2D or 3D) based on the provided dimensionality Dim. For Dim == 2, it resolves to voro::voronoicell_neighbor_2d, and for Dim == 3, it resolves to voro::voronoicell_neighbor.

Template Parameters
DimDimensionality of the Voronoi tessellation. Must be either 2 or 3.

Computes the signed distance between a point and a shape defined by a distance function.

This function takes a point and a distance function (which conforms to the DistanceExpression interface) and computes the distance from the point to the shape represented by the distance function. The result is stored in a buffer and returned as a std::array of distances.

Template Parameters
FloatTypeThe type of floating-point value used for distance computation (e.g., float, double).
DimThe dimensionality of the space (e.g., 2 for 2D, 3 for 3D).
DThe type of the distance function, which must inherit from DistanceExpression<FloatType, Dim, D>.
Parameters
pointThe point for which the distance to the shape is to be computed.
distance_functionThe distance function object representing the shape.
Returns
A std::array of distances, where the number of elements is determined by the size of the distance buffer.

Computes a Voronoi tessellation in either 2D or 3D space.

This function is a general wrapper that selects between 2D and 3D Voronoi tessellation based on the template parameter Dim. It calls either voronoi2D or voronoi3D internally.

Template Parameters
DimDimensionality of the Voronoi tessellation. Must be either 2 or 3.
Parameters
bboxA bounding box defined by two corner points. The array should contain two Point<double, Dim> values representing the corners of the bounding box.
pointsA vector of Point<double, Dim> representing the coordinates of the points for which the Voronoi diagram is to be computed.
cell_callbackA function callback that is invoked for each generated Voronoi cell. The callback function takes two parameters: the generated VoronoiCell<Dim> and an integer representing the ID of the cell.

Computes a 2D Voronoi tessellation for a given set of points within a bounding box.

This function utilizes the Voro++ library to generate Voronoi cells in a 2D space. Each generated Voronoi cell is passed to the provided callback function for further processing.

Parameters
bboxA bounding box defined by two corner points. The array should contain two Point<double, 2> values representing the lower-left and upper-right corners.
pointsA vector of Point<double, 2> representing the coordinates of the points for which the Voronoi diagram is to be computed.
cell_callbackA function callback that is invoked for each generated Voronoi cell. The callback function takes two parameters: the generated voro::voronoicell_neighbor_2d and an integer representing the ID of the cell.
