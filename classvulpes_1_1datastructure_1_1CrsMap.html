Detailed Description

template<typename DataType, typename IndexType, IndexType RowSize = IndexType(-1)>
class vulpes::datastructure::CrsMap< DataType, IndexType, RowSize >
Template Parameters
DataTypeThe type of data to be stored in the CRS map.
IndexTypeThe type used for indices in the CRS map.
RowSizeSize of each row. If set to -1, the row size is dynamic.

This class represents a Compressed Row Storage (CRS) structure, which works like a vector of vectors for efficient storage. If DataType is a std::pair<IndexType, double>, it models a CRS matrix. The row size can either be fixed or dynamic based on the RowSize template argument.

Constructor that initializes a CrsMap by moving the provided data and indices.

This constructor takes ownership of the provided data and indices vectors by moving them into the internal storage. This allows efficient initialization without copying the vectors, especially when compressing a CrsMapUncompressed

Parameters
dataA rvalue reference to a vector containing the data elements, typically representing non-zero values in a compressed row storage (CRS) format. The vector will be moved into the internal storage.
indicesA rvalue reference to a vector containing the row indices in the CRS format. The vector will be moved into the internal storage.
Note
After calling this constructor, the passed data and indices vectors will be in a valid but unspecified state, as their contents will have been moved into the CrsMap instance.
