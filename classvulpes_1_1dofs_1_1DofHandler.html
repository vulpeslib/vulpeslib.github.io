Detailed Description

template<typename MeshType>
class vulpes::dofs::DofHandler< MeshType >

Compute and distribute the DoFs and sparsity pattern associated with a mesh and a certain finite element space.

This class implements the tools to distribute and compute the DoFs and sparsity pattern of a mesh and finite element space both in serial and in parallel. The class distinguishes between the two cases by checking the boolean value of MeshType::parallel(). The DofHandler also manages the renumbering of the DoFs to minimize the bandwidth of the sparsity pattern (to enhance cache locality and reduce the condition number of the final matrix). If the mesh is parallel the class automatically applies a preliminary renumbering in order to minimize intra-process communications at assebly-time.

Note
: before using the class call the method finalize to apply the cell_to_cell_dof permutation to each of the rows in cell_to_dof. This is not automatically done since the user may want to apply a renumbering to the dofs.

Compute owned cell dofs and boundary dofs.

Find which are the DoFs that belong to cell that are owned by this proc. Moreover, find which are the DoFs that lie on the boundary on the partition, that is, DoFs that belong both to ghost cells and owned cells.
