Detailed Description

template<typename T>
class vulpes::polynomial::Polynomial< T >

Template class to represent and operate on polynomials.

This file contains the implementation of a polynomial class template that supports various operations, including addition, subtraction, multiplication, division, differentiation, integration, and more.

◆ evaluate() [3/4]

template<typename T >
T vulpes::polynomial::Polynomial< T >::evaluate ( const T &  x) const
inline

Evaluates the polynomial at a given point using Horner's method.

Horner's method is used to efficiently evaluate the polynomial of degree \( n \) at a given point \( x \). The polynomial:

\[ P(x) = a_n x^n + a_{n-1} x^{n-1} + \dots + a_1 x + a_0 \]

is rewritten using Horner's rule as:

\[ P(x) = (((a_n x + a_{n-1}) x + a_{n-2}) x + \dots + a_1) x + a_0 \]

This reduces the number of multiplications needed to evaluate the polynomial.

Parameters
xThe point at which to evaluate the polynomial.
Returns
The value of the polynomial at the point x.
