Detailed Description

template<std::size_t Dim, typename FeSpaceType, typename DofHandlerType>
class vulpes::fe::FeGradProjectionExpr< Dim, FeSpaceType, DofHandlerType >

Physical gradient of a DG field given its DOF coefficient vector.

Originally introduced to support H1 semi-norm error computation (int_K |grad(u_h - u_ref)|^2 dK). Also suitable when the gradient of a projected coefficient field is needed as a composable expression.

Evaluates grad(u_h)(x_q) = sum_k c_k * grad(phi_k)(x_q) at every quadrature point using the precomputed physical gradients from fe_values.getGradEval(). Requires UpdateGradients.

OutputType is Eigen::Vector<double,Dim>. computeOrder returns order-1 (gradient lowers polynomial degree by one). The expression is a pure coefficient node (is_bilinear = IsNothing).

Use the factory fe::grad_proj to construct:

auto grad_p = fe::grad_proj(p_vec, fe_space, dof_handler);
// curl of a 2D vector field (d_x(q) - d_y(p)):
auto curl = fe::spatial_component<0>(fe::grad_proj(q_vec, fe_space,
dof_handler))
- fe::spatial_component<1>(fe::grad_proj(p_vec, fe_space,
dof_handler));
