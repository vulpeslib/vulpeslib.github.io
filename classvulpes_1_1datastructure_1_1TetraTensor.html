Detailed Description

template<bool Triangular, bool Tensor, typename DataType, typename IndexType, typename StorageType = std::vector<DataType>>
class vulpes::datastructure::TetraTensor< Triangular, Tensor, DataType, IndexType, StorageType >

A class that represents a tetrahedral matrix or tensor, with optional support for higher-dimensional tensor-like data.

The TetraTensor class stores data in either a tetrahedral structure (if Triangular is true) or in a cubic structure (if Triangular is false). Namely it offers a coalescent representation for data when you know that the indexes of the tensor i, j, k satisfy i + j + k < tri_dim. It can also store fourth-order tensor data based on the template parameters.

Template Parameters
TriangularSpecifies whether the tensor is tetrahedral (if true).
TensorSpecifies whether the tensor includes additional dimensions.
DataTypeThe data type of elements stored in the tensor.
IndexTypeThe data type for indexing the tensor.
StorageTypeThe container where to save the data. By default an std::vector, other used options are a const pointer, indicating that this is an not-owning view of some data
