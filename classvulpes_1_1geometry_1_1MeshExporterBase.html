Detailed Description

template<typename MeshDerived>
class vulpes::geometry::MeshExporterBase< MeshDerived >

Base class for exporting a mesh to VTK file formats.

This template class provides the core functionality for converting a mesh data structure into one or more VTK unstructured grids and saving them to a file. It handles the common tasks of writing VTU or VTK files.

Template Parameters
MeshDerivedThe derived mesh class, which must be derived from MeshBase

Saves the mesh data to a file.

This method writes the internal VTK grid data to the specified filename. It supports .vtu (VTK Unstructured Grid XML) and .vtk (legacy VTK) file formats.
