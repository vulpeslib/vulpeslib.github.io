Detailed Description

template<typename MeshType, typename FeExpressionTuple, typename FeExpressionFaceTuple = std::tuple<>>
class vulpes::assemble::WeakFormIntegratorVem< MeshType, FeExpressionTuple, FeExpressionFaceTuple >

Class to perform numerical integration of weak form integrals for VEM problems.

Warning
This class is just a sketch of how VEM integrators should work. In particular, the main ideas are:
  • The actual integration is done by another integrator (QFree, Simplexify, Gauss, ...)
  • The weak forms to integrate are the one passed by the user, plus the one requested by the VEM Finite Element space (e.g. the integrals of each pair of scaled monomials).
  • The VEM finite element space is in charge of computing the "extraMatrices" by using the requested extra integrals. For instance, the matrix "D" is computed by the VEM space using the mass matrix
