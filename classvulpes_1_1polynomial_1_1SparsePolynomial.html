Detailed Description

template<std::size_t Dim, typename IndexType, typename CoefType, bool IsScaled>
class vulpes::polynomial::SparsePolynomial< Dim, IndexType, CoefType, IsScaled >

Template class to represent and operate on polynomials in nD.

The class is able to handle also polynomials that are sum/difference of scaled monomials, this comes particularly handy in VEM.

The monomials are stored in a SmallVector, ensuring static allocation for a small number of monomials.

This file contains the implementation of a polynomial class template that supports various operations, including addition, subtraction, multiplication, division, differentiation, integration, and more.

Note
The class is optimized for sparse storage, meaning that it is adapt to cases where most of the coefficients are zero.
