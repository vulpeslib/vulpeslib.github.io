template<typename T >
std::vector< Polynomial< T > > vulpes::polynomial::buildLegendrePolynomials ( std::size_t  deg)

Builds a set of Legendre polynomials up to a given degree.

The Legendre polynomials \( P_n(x) \) are generated using the recurrence relation (Bonnet formula):

\[ P_{n+1}(x) = \frac{(2n+1) \cdot x \cdot P_n(x) - n \cdot P_{n-1}(x)}{n+1} \]

with the initial conditions:

\[ P_0(x) = 1, \quad P_1(x) = x \]

Template Parameters
TThe type of the coefficients in the polynomials.
Parameters
degThe degree up to which Legendre polynomials should be built.
Returns
A vector containing the Legendre polynomials from degree 0 to deg.
