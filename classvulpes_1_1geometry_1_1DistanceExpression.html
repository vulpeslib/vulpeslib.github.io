Detailed Description

template<typename FloatType, int32_t Dim, typename Distance>
class vulpes::geometry::DistanceExpression< FloatType, Dim, Distance >

Template class representing a generic signed distance expression.

This template class defines the interface for various distance functions via CRTP. It provides static methods and operators to calculate the distance between a given point and a shape, and to compute the bounding box (bbox) of that shape. The distance expression keeps in "memory" the distances with each shape it is composed of (e.g. in the intersection of a rectange and a circle the output distance gives the distance with respect to each segment of the rectange and the circle)

Template Parameters
FloatTypeThe type of floating-point value (e.g., float, double).
DimThe dimensionality of the space (e.g., 2D or 3D).
DistanceThe specific distance function that will be used (e.g., RectangleDistance, CircleDistance).
