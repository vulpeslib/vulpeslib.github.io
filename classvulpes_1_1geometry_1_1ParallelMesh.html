Detailed Description

template<typename Mesh>
class vulpes::geometry::ParallelMesh< Mesh >

This class wraps any serial mesh class and adds suitable attribute needed to distributes it in parallel.

There are the following ways to build a parallel mesh:

  • parellize an already build mesh by proc 0 by taking its ownership
