Extracts the dual subgraph corresponding to a subset of vertices.

Given a dual graph for an entire mesh and a list of vertex indices, this function constructs and returns the induced subgraph containing only the specified vertices and the edges between them. The vertices in the resulting subgraph are re-indexed to form a contiguous range [0, N-1], where N is the number of vertices in the subset.

Template Parameters
index_tThe integral type used for graph indices.
Parameters
dual_mapThe dual graph of the entire mesh, represented as a CrsMap.
subset_idxsA vector of vertex indices defining the subgraph to extract.
Returns
A new CrsMap representing the extracted subgraph.

Re-types a CRS map to use a different integral type for indices and values.

This function converts a CrsMap from a source integral type Src to a destination integral type Dst. If Dst and Src are the same, the input map is returned unchanged. Otherwise, it constructs a new CrsMap by performing an element-wise cast on both the data and index arrays.

Template Parameters
DstThe destination integral type for the new map's indices and values.
SrcThe source integral type for the input map's indices and values.
Parameters
mapThe input CrsMap with data and indices of type Src.
Returns
The original map if Dst is the same as Src, otherwise a new CrsMap<Dst, Dst> with casted elements.
Warning
This operation may result in narrowing if values or indices in the source map do not fit within the range of the Dst type.
Note
Complexity is Linear in the number of elements: $O(|map.data()| + |map.indices()|)$.

template<typename VecType >
void vulpes::geometry::internal::checkEmptySets ( const VecType &  labels)

Checks for and warns about empty classes in a label vector.

This function inspects a vector of labels and determines the number of classes as max(labels) + 1. It then checks if every class ID in the range [0, max(labels)] is present at least once. A warning message is printed to std::cerr for each class that has no members.

Template Parameters
VecTypeA random-access container type that supports size() and operator[], such as std::vector.
Parameters
labelsA container of partition labels. Values are expected to be non-negative.
Note
complexity $O(N + M)$, where $N$ is the number of labels and $M$ is the maximum label value.
This function has no side effects other than potentially printing diagnostic messages to standard error.

◆ removeEmptySets()

template<typename index_t >
index_t vulpes::geometry::internal::removeEmptySets ( std::vector< index_t > &  labels)

Compresses labels to a contiguous range [0, N-1].

This function modifies a vector of labels in-place, remapping them so that the set of unique labels becomes a dense range [0, N-1], where N is the total number of unique labels originally present. The relative ordering of labels is preserved.

For example, an input of {3, 0, 3, 5, 0} would be modified to {1, 0, 1, 2, 0}, and the function would return 3.

Template Parameters
index_tThe integral type of the labels.
Parameters
[in,out]labelsThe vector of labels to be modified.
Returns
The number of unique labels, N.
