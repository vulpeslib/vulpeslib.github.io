Detailed Description

template<typename T, bool IsSpaceP = true>
class vulpes::polynomial::Polynomial2D< T, IsSpaceP >

Template class to represent and operate on polynomials in 2D.

This file contains the implementation of a polynomial class template that supports various operations, including addition, subtraction, multiplication, division, differentiation, integration, and more.

Template Parameters
IsSpacePselect if the polynomial belongs to P space or Q space for optimized storage
Note
The class is optimized for dense storage, meaning that it is adapt to cases where most of the coefficients are different from zero. Once the storage size is allocated when the class is constructed it cannot be changed.
