Detailed Description

template<typename FloatType>
class vulpes::geometry::dist2d::RectangleDistance< FloatType >

Signed distance function for a rectangle in 2D space.

This class computes the signed distance between a point and a rectangle defined by two corner points.

Template Parameters
FloatTypeThe type of floating-point value.
