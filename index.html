Introduction

Vulpes (VirtUaL and Discontinuous Polytopal ElementS method) is an open-source library.

Contribute

The Vulpes team welcomes contributions at all levels: bugfixes, code improvements, simplifications, improved documentation, new examples and so on. If you plan on contributing to Vulpes check out our Developer's guide
