Detailed Description

template<typename FloatType, typename IndexType, IndexType OuterDim, IndexType SelfDim, typename Derived>
class vulpes::geometry::MeshBase< FloatType, IndexType, OuterDim, SelfDim, Derived >

A base class for creating and managing mesh data structures.

This class provides the fundamental storage and connectivity information for a mesh, such as vertices, cells, and faces. It uses a common programming pattern called CRTP (Curiously Recurring Template Pattern), which allows it to be extended by a Derived class with more specific features.

It defines the relationships between different mesh parts (e.g., which vertices form a cell, which cells are neighbors).

Template Parameters
FloatTypeThe data type for floating-point numbers (e.g., float or double).
IndexTypeThe data type for indices (e.g., int or std::size_t).
OuterDimThe dimension of the space the mesh lives in (e.g., 3 for a surface in 3D space).
SelfDimThe dimension of the mesh itself (e.g., 2 for a 2D mesh, 3 for a 3D mesh).
DerivedThe actual mesh class that inherits from this base (CRTP).
