Detailed Description

Performs triangulation of a 2D cell using the Triangle library.

This class handles the generation of a triangulation for a given 2D cell and provides access to the resulting points and triangles. It leverages the Triangle library [https://www.cs.cmu.edu/~quake/triangle.html] under the hood for computational geometry operations.
