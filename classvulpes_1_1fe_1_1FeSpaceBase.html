Detailed Description

template<std::size_t Dim, typename FloatType, typename DofPolicy, typename FeSpace>
class vulpes::fe::FeSpaceBase< Dim, FloatType, DofPolicy, FeSpace >

Class describing the template for a FeSpace.

A FeSpace class should define

  • a basis of polynomial in arbitrary float precision
  • a basis of polynomial in 32bit or 64bit float precision
  • define on which types of elements FeType it can work with
  • define how many dofs are on any geometrical entity (vertex, edge, ...)
  • define the dof interaction distance (at which cell distance, in the dual map, two dofs may have non-zero integrals)
  • polynomial order

There are 11 avaiable FE spaces:

  • Lagrangian/Spectral on Simplex in both Continuous and Discontinuous versions
  • Lagrangian/Spectral on Tensors in both Continuous and Discontinuous versions
  • Modal on Simplex in discontinuous version
  • Modal on Tensor in discontinuous version
  • VEM
