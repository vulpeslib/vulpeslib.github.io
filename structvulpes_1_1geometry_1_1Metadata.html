Detailed Description

template<typename FloatType, typename IndexType>
struct vulpes::geometry::Metadata< FloatType, IndexType >

A structure to store global metadata.

Metadata represents global mesh properties that are not tied to specific mesh entities. For example: mesh dimensions, physical parameters, periodicity information, etc.

Template Parameters
FloatTypeThe type used for floating-point metadata values.
IndexTypeThe type used for integer metadata values.
