Detailed Description

template<std::size_t K, std::size_t Dim, typename E>
class vulpes::fe::FeExpressionSpatialComponent< K, Dim, E >

Extract a single scalar spatial component at compile-time index K from a vector-output FE expression.

Works for all expression types (IsNothing, IsLinearTest, IsLinearTrial, IsBilinear) whose evaluate() returns a type supporting operator[K]. Returns the K-th spatial component as a double at every quadrature point.

Typical uses:

auto d_x_p = fe::spatial_component<0>(fe::grad_proj(p_vec, fe_space,
doh)); auto d_x_ph = fe::spatial_component<0>(fe::grad(phi)); // d/dx of
test fn auto d_y_ph = fe::spatial_component<1>(fe::grad(phi)); // d/dy of
test fn auto nx = fe::spatial_component<0>(fe::normal(fe_space)); // x
normal

Compute the polynomial associated to the expression for basis i and j and block bi and bj.

Parameters
fe_valuesFinite element values at quadrature points
jIndex of the trial function basis
iIndex of the test function basis
biBlock index of the trial function
bjBlock index of the test function
Returns
Matrix of polynomials associated to the expression

We use the convention that in the result, each row corresponds to a spatial direction (this extends also to tensors, by unrolling them), while each column corresponds to a addendum of the expression requiring a particular pullback/pushforward transformation. These coefficients are computed by the evaluatePolyCoef method.
