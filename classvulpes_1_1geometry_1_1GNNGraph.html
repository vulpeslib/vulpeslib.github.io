Detailed Description

Graph wrapper with Torch tensors for features and edges.

Assumes SAGEConv-compatible dtypes (float32 features, int64 indices). The topology is stored as a compressed sparse row map (CRS), and a 2 x nnz edge_index tensor is built for Torch models.

Build the Torch COO edge tensor from the CRS map.

GNNGraph vulpes::geometry::GNNGraph::getSubGraph ( const IndVec &  subset_idxs)

Induced subgraph on a set of node indices.

Builds a new GNNGraph containing only nodes in subset_idxs, with features row-selected, edges filtered to pairs fully inside the subset, and node ids reindexed densely from 0..m-1 (preserving the order of subset_idxs).

Parameters

Returns
A new GNNGraph restricted to the subset.
