Detailed Description

template<std::size_t Dim>
class vulpes::fe::FeValues< Dim >

This class works as a cache table for computing efficiently the value of the basis function and its derivatives for each cell.

The values of the basis function and its derivatives saved for the reference element and the values updated for each cell. Computing the values on the reference cell is usually expensive since the polynomial of the basis function is saved in arbitrary precision. The update is usually fast and involves computing some properties of the mapping function. We minimize the computations by indicating what needs an update using UpdateFlags.

In DG elements it happens that a cell needs the values of the neighbour. To accomodate this necessity the class owns a pointer to the FeValues of the neighbour.
